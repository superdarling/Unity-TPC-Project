#  Unity TCP Control Project

##  项目简介
本项目基于 Unity 引擎开发，实现 PC 端与移动端之间的实时通信与远程控制系统。

移动端通过 UI 控件发送控制指令，PC 端执行角色行为（移动、动画、贴图、高亮等），并将执行结果以日志形式回传，实现双向通信与数据可视化。

---

##  技术栈
- Unity（2D/3D）
- C#
- TCP Socket
- UGUI / TextMeshPro

---

##  核心功能
-  移动端远程控制角色移动
-  动画切换控制
-  对象高亮与交互
-  实时日志回传与分类显示
-  双端通信（客户端 / 服务端）

---

##  
- 自主设计 TCP 通信协议，实现稳定数据传输
- 模块化架构（UI / 控制 / 网络分层）
- 日志系统支持分类筛选与动态展示
- 支持跨平台控制（PC + Mobile）

---


##  运行方式

1. 打开 Unity 项目
2. 联通测试时TCPClient脚本下的ip需要改为服务器IP（电脑的IP），apk文件用的还是本机IP，无法直接使用
3. 运行 PC 端场景
4. 启动移动端控制界面
5. 建立 TCP 连接
6. 开始控制角色
